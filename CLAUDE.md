# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## プロジェクト概要

製造・エンジニアリング業務向けの生成AIプロンプト活用プレゼンテーションシステム。HTMLベースのスライド形式で、プロンプトエンジニアリングの基礎から実践事例までを日本語で解説。

## 主要ファイル構成

- `index.html`: プレゼンテーションのメインHTMLファイル（全スライドを含む）
- `js/presentation.js`: スライドナビゲーション、キーボード/タッチ操作、コピーボタン機能
- `css/style.css`: Atlassian Design Systemベースのスタイリング
- `修正案_ヤマハモーターエンジニアリング向け.md`: プレゼンテーション修正案の詳細ドキュメント
- `設計レビュー議事録_YKシリーズ次期モデル.md`: デモ用のサンプル議事録データ

## アーキテクチャ

### スライドシステム
- 各スライドは`.slide`クラスの`<div>`要素で定義
- JavaScriptによる`.active`クラスの付け替えで表示切替
- CSSアニメーション（`slideIn`）でスライド遷移を演出

### ナビゲーション機能
- **キーボード**: 右矢印/スペース（次へ）、左矢印（前へ）、Home（最初）、End（最後）
- **タッチ**: 左右スワイプでスライド移動
- **ボタン**: 画面下部の前へ/次へボタン
- プログレスバーとスライド番号で進捗表示

### コードブロック機能
- `<pre><code>`ブロックを自動検出して`.code-container`でラップ
- 各コードブロックに「コピー」ボタンを動的追加
- Clipboard APIを使用した1クリックコピー機能（フォールバック付き）

## プレゼンテーション内容の編集

### スライドの追加・削除
1. `index.html`内の`.presentation-container`に新しい`.slide`要素を追加
2. JavaScriptが自動的にスライド総数を検出
3. スライド番号とプログレスバーは自動更新

### スライドの種類
- **タイトルスライド**: `.slide.active.title-slide`（中央揃え、大きなタイトル）
- **通常スライド**: `.slide`（左揃え、見出し+コンテンツ）
- **比較スライド**: `.comparison`グリッド（Before/Afterの対比）

### コンテンツ要素
- **強調ボックス**: `.highlight-box`（左に青い縦線）
- **カード**: `.card`（ホバーで浮き上がる効果）
  - バリエーション: `.card.success`（緑）、`.card.warning`（黄）、`.card.error`（赤）
- **機能グリッド**: `.feature-grid`（自動レスポンシブグリッド）
- **サマリーボックス**: `.summary-box`（グラデーション背景）

## 開発コマンド

### プレゼンテーションの起動
```bash
# ローカルサーバーで起動（推奨）
python -m http.server 8000
# ブラウザで http://localhost:8000 を開く

# または任意のHTTPサーバー
npx serve
```

### ファイル編集後の確認
- HTMLファイルを編集後、ブラウザをリロード
- CSSやJSはキャッシュをクリアして確認（Cmd+Shift+R / Ctrl+Shift+R）

## プレゼンテーション実施時の注意点

### 操作方法
- フルスクリーン表示推奨（F11キー）
- スペースキーまたは右矢印でスライド送り
- コードブロックの「コピー」ボタンはプレゼン中に実演可能

### デモ用データ
- `設計レビュー議事録_YKシリーズ次期モデル.md`: ファイルアップロードデモで使用
- 画像ファイル（`image.png`, `image-1.png`等）: 実際の出力例のスクリーンショット

### カスタマイズが必要な箇所
- スライド1: タイトル、日付
- スライド番号の総数（JavaScriptが自動計算）
- 会社ロゴ（`/img/c-table_logo.png`）
- デモ用の実データ（議事録、技術仕様書等）

## 技術的制約

- **PC専用設計**: モバイル対応は最小限（タッチスワイプのみ）
- **モダンブラウザ必須**: Clipboard API、CSS Grid、ES6対応
- **オフライン対応なし**: Font AwesomeはCDN経由で読み込み
- **印刷非対応**: スライド形式のため紙媒体には不向き

## トラブルシューティング

### スライドが表示されない
- ブラウザのコンソールでJavaScriptエラーを確認
- `.slide`要素の数とJavaScriptの`totalSlides`が一致するか確認

### コピーボタンが動作しない
- Clipboard APIはHTTPS環境または`localhost`でのみ動作
- HTTPサーバー経由でアクセスしているか確認

### スタイルが崩れる
- Font Awesome CDNが読み込めているか確認
- ブラウザキャッシュをクリア
- CSSファイルのパスが正しいか確認（`css/style.css`）
